Class: Room

Room(disable, turns, status, bid, bidlog, bidWinner, playerHands, players, spectators, clients, turnStatus, winner, finalScore)

new Room(disable, turns, status, bid, bidlog, bidWinner, playerHands, players, spectators, clients, turnStatus, winner, finalScore)

Creates a class that stores the room state as follows:
Parameters:
Name Type Description
disable boolean When set to true, stops all players from playing any cards by making cards unclickable.
turns Number Tracks which player turn to move during bidding and playing phase.
status string Tracks current game phase. Possible values are: "setup", "start", "bid", "allPass", "selectPartner", "play", "gameOver".
bid Number Store the current bid.
bidlog Array Store all historical bid.
bidWinner Object Store details of user who won the bid, and details about the winning bid and partner.
playerHands Object Store Cards Object in each player hand.
players Object Hash table of role to player name.
spectators Array List of all spectators name.
clients Number Number of clients connected to the room.
turnStatus Object Tracks board status like cards played, trumpBroken status.
winner Array Store the names of the winners.
finalScore Number Store the number of tricks that winner has won.
